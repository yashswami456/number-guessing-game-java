# Number Guessing Game 🎮

A Java-based Number Guessing Game project.

## Features
- Random number generation
- Multiple rounds
- Score system
- Limited attempts
- High/Low hints

## Technologies Used
- Java
- OOP Concepts
- IntelliJ IDEA

## How to Run
1. Open project in IntelliJ IDEA
2. Run Main.java
3. Start guessing the number
